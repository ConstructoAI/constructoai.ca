Anticipons Vos Objections

Qu'est-ce qui est inclus dans l'abonnement?

L'abonnement de 139.99$/mois inclut: accès complet aux 27+ modules ERP (CRM, Devis, Comptabilité, TimeTracker, Inventaire, etc.), 57 profils d'experts IA spécialisés construction, espace entreprise privé et sécurisé (multi-tenant PostgreSQL), support technique, mises à jour automatiques et hébergement cloud sur Render. Utilisateurs illimités!

Mes données sont-elles sécurisées?

Absolument. Chaque entreprise dispose d'un espace 100% isolé grâce à notre architecture multi-tenant PostgreSQL. Vos données sont protégées par hachage bcrypt (12 rounds), rate limiting contre les attaques brute force (5 tentatives max/30 min), sessions sécurisées avec déconnexion automatique après 30 min d'inactivité, et HTTPS forcé en production. Backups automatiques quotidiens inclus.

Combien d'utilisateurs peuvent utiliser l'ERP?

Illimité! Vous pouvez créer autant de comptes utilisateurs que nécessaire pour votre entreprise sans frais supplémentaires. Notre architecture PostgreSQL supporte 1000+ utilisateurs simultanés avec un temps de réponse moyen inférieur à 200ms.

Dois-je fournir ma propre clé API pour l'IA?

Non! La clé API Claude (Anthropic) est incluse dans votre abonnement de 139.99$/mois. Vous avez accès immédiatement aux 57 experts IA spécialisés, à la recherche web et à l'analyse de documents sans aucune configuration supplémentaire.

Combien coûte vraiment Constructo AI?

L'ERP complet coûte 139.99$/mois après 7 jours d'essai (carte de crédit requise pour l'essai). Aucun frais si vous annulez avant la fin de l'essai. Cartes acceptées: Visa, Mastercard, Amex, Visa Desjardins. Le module SEAOP (appels d'offres publics) est 100% gratuit, sans carte requise.
