Fonctionnalités du module de soumission

Base de données de prix construction

Accédez à une base de données complète des prix de matériaux et main-d'oeuvre pour le marché québécois. Les prix sont mis à jour régulièrement pour refléter les conditions du marché. Vous pouvez aussi importer et gérer vos propres listes de prix.

Calcul automatique des coûts

Le système calcule automatiquement les coûts totaux incluant les matériaux, la main-d'oeuvre, les frais généraux et vos marges. Les taxes TPS/TVQ sont calculées automatiquement selon les règles en vigueur.

Modèles de soumission personnalisables

Créez des modèles réutilisables pour vos types de projets récurrents. Personnalisez l'apparence de vos documents avec votre logo, couleurs et mentions légales.

Suivi et relance automatique

Suivez le statut de chaque soumission : envoyée, vue, en négociation, acceptée ou refusée. Configurez des rappels automatiques pour relancer les clients qui n'ont pas répondu.

Analyse de performance

Visualisez votre taux de conversion, le montant moyen de vos soumissions et identifiez les types de projets où vous performez le mieux.

Questions fréquentes

Comment faire une soumission de construction?
Avec Constructo AI, vous créez une soumission en quelques étapes : sélectionnez le type de projet, ajoutez les items depuis notre base de données de prix, ajustez vos marges avec l'aide de l'IA, et générez un document PDF professionnel à envoyer au client.
Combien coûte un logiciel de soumission construction?
Constructo AI offre un module de soumission complet inclus dans son forfait à 79,99$/mois, avec tous les autres modules ERP. Certains logiciels spécialisés coûtent entre 50$ et 200$/mois pour les soumissions seulement.
Peut-on importer ses prix dans le logiciel?
Oui, Constructo AI permet d'importer vos propres listes de prix via Excel ou CSV. Vous pouvez aussi utiliser notre base de données de prix du marché québécois comme référence.
Le logiciel fonctionne-t-il sur mobile?
Oui, Constructo AI est une application web qui fonctionne sur tous les appareils. Vous pouvez créer et envoyer des soumissions depuis votre téléphone ou tablette directement sur le chantier.
Puis-je personnaliser l'apparence de mes soumissions?
Absolument. Vous pouvez ajouter votre logo, choisir vos couleurs, personnaliser le texte d'introduction et les conditions générales. Vos soumissions auront un look professionnel à votre image.
